# initWithColor:

初始化无限范围的图像，其整个内容是指定的颜色。

| SDK | Version |
|:---:|:---:|
| iOS | 5.0+ |
| macOS | 10.5+ |
| tvOS | 9.0+ |

---

## 声明

```objective-c
/* Objective-C */
- (instancetype)initWithColor:(CIColor *)color;
```

```swift
/* Swift */
init(color: CIColor)
```

---

## 参数

* **color**

    一个颜色对象。

---

## 返回值

初始化的图像对象。
