# draw(withFrame:in:)

绘制接收者的边框，然后绘制`Cell`的内部。

## 定义

```swift
func draw(withFrame cellFrame: NSRect, 
               in controlView: NSView)
```

## 参数

* **cellFrame**

    接收者的边界矩形。

* **controlView**

    管理`Cell`的`Control`对象。

## 说明

此方法在当前聚焦的`View`中绘制`Cell`，该`Cell`可能与传入的`controlVie`w不同。但是，建议不要利用此行为。
